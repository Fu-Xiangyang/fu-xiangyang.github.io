1.1 测试的含义

信息蕴含于信号之中，信号是传输信息的载体。

测试四要素：被测对象，计量单位，测试方法，测量误差

测量是指将一个被测量与一个预定标准之间进行定量比较，从而获得被测对象的数值结果，即以确定被测对象的量值为目的的全部操作。

1.2.2 测量方法的分类

  1. 按测量方法分类（直接测量、间接测量、组合测量）
  2. 按被测量随时间变化的快慢分类（静态测量、动态测量）
  3. 按测量条件分类（等精度测量、非等精度测量）
  4. 按测量方式分类（偏差式测量、零位式测量、微差式测量）
  5. 按测量设备与物质的接触情况分类（接触式测量、非接触式测量）

第二章 测试系统的基本特性

2.1 测试系统概述

2.1.1测试系统基本概念

测试系统是指为完成某种物理量的测量而由某种或多种变换特性的物理装置构成的总体。

测试装置自身的物理结构所决定的测试系统对信号传递变换的影响特性称为”测试系统的传递特性“，简称”系统的传递特性“或”系统的特性“。

系统分析中的三类问题：

  • 当输入、输出是可测量的(已知)，可以通过它们推断系统的传输特性。（系统辨识）
  • 当输入特性已知，输出可测量，可以通过它们推断导致该输出的输入量。（反求）
  • 如果输入和系统特性已知，则可以推断和估计系统的输出量。（预测）

2.1.2 理想测试系统——线性时不变系统

​ 我们希望测试系统的输出信号能准确地反映被测量，即理想的测试系统应该具有单值的、确定的输入/输出关系，而且输出与输入之间以线性关系为最佳。

第三章 传感器及其应用

3.1 概论

3.1.1 传感器的定义

能感受规定的被测量并按照一定的规律转换成可用输出信号的器件或者装置。

  1. 传感器是测量装置，能完成检测技术
  2. 一个指定的传感器只能感受或响应规定的被测量，被测量既可以是电量也可以是非电量
  3. 按一定规律转换成易于传输和处理的信号，而且这种规律是可复现的
  4. 传感器的输出信号不仅承载着待测的原始信号，而且能够被传输并成为便于后继检测环节接收和进一步处理的信号形式

3.1.2 传感器的组成

传感器通常由敏感元件，转换元件，信号调节和转换电路以及辅助电源组成

  1. 敏感元件指传感器中能直接感受或响应被测量的部分
  2. 转换元件指传感器中能将敏感元件感受或响应的被测量转换成适用于传输和测量的电信号部分
  3. 信号调节和转换电路可以对微弱电信号进行放大，运算调制等
  4. 辅助电源可以为转换元件以及信号调节和转换电路提供必要的动力

3.1.3 传感器的分类

  1. 按被测物理量分类
  2. 按传感器工作原理
  3. 按能量传递方式
    • 能量控制型
    • 能量转换型
  4. 按内部物理结构
    • 结构型
    • 物性型
  5. 按输出信号特征
    • 模拟式
    • 数字式
    • 开关式

3.1.4 传感器的选用原则

考虑问题：

  • 量程的大小
  • 传感器的大小
  • 安装方式
  • 测量方式
  • 信号的引出方式
  • 传感器的来源

考虑性能指标

  • 灵敏度的选择
  • 频率响应特性
  • 线性范围
  • 稳定性
  • 精度
