一. OpenCV简介

1. 图像处理简介

图像是什么？

图像是人类视觉的基础，是自然景物的客观反映，是人类认识世界和人类本身的重要源泉。“图”是物体反射或透射光的分布，“像”是人的视觉系统所接受的图在人脑中所形版的印象或认识，照片、绘画、剪贴画、地图、书法作品、手写汉学、传真、卫星云图、影视画面、X光片、脑电图、心电图等都是图像。

模拟图像和数字图像

模拟图像又称连续图像，它通过某种物理量（如光、电等）的强弱变化来记录图像亮度信息，所以是连续变换的。

二. OpenCV基本操作

1. 图像的基础操作

1.1 图像的IO操作

  1. 读取图像

    API:

    cv.imread()

    参数：

    • 要读取的图像

    • 读取方式的标志

      • cv.IMREAD*COLOR : 以彩色模式加载图像（默认参数）
      • cv.IMREAD*GRAYSCALE : 以灰度模式加载图像

      • cv.IMREAD_UNCHANGED : 包括alpha通道的加载图像模式

        可以使用1、0、-1来替代上面三个标志

    • 参考代码

      import numpy as np
import cv2
img = cv2.imread('1.jpg',0)

      如果加载的路径有错误，不会报错，会返回一个None值

  2. 显示图像

    API:

    cv.imshow()

    参数：

    • 显示图像的窗口名称，以字符串类型表示

    • 要加载的图像

      在调用显示图像的API后，要调用cv2.waitKey()给图像绘制留下时间，否则会出现无响应情况，并且图像无法显示出来

      另外也可以使用matplotlib对图像进行显示

      参考代码

      cv2,imshow('image',img)
cv.waitKey(0)

plt.imshow(img[:,:,::-1])

  3. 保存图像

    API:

    cv.imwrite()

    参数：

    • 文件名，要保存到哪里

    • 要保存的图像

      参考代码

      cv2.imwrite('1.jpg',img)

1.2 绘制几何图形

  1. 绘制直线

    cv2.line(img,start,end,color,thickness)

    参数：

    • img：要绘制直线的图像
    • start,end：直线的起点和终点
    • color：线条的颜色
    • thickness：线条宽度

  2. 绘制圆形

    cv2.circle(img,centerpoint,r,color,thickness)

    参数：

    • img：要绘制圆形的图像
    • centerpoint, r：圆心和半径
    • color：线条的颜色
    • thickness：线条宽度，为-1时生成闭合图案并填充颜色

  3. 绘制矩形

    cv2.rectangle(img,leftupper,rightdown,color,thickness)

    参数：

    • img：要绘制矩形的图像
    • leftupper,rightdown：矩形的左上角和右下角坐标
    • color：线条的颜色
    • thickness：线条宽度

  4. 向图像中添加文字

    cv2.putText(img,text,station,font,fontsize,color,cv.LINE_AA)

    参数：

    • img：图像
    • text：要写入的文本数据
    • station：文本的放置位置
    • font：字体
    • fontsize：字体大小

  5. 展示：生成一个全黑的图像，在里面绘制图像并添加文字

    import numpy as np
import cv2
import matplotlib.pyplot as plt

img = np.zeros((512,512,3),np.uint8)

cv2.line(img,(0,0),(511,511),(255,0,0),5)
cv2.rectangle(img,(384,0),(510,128),(0,255,0),3)
cv2.circle(img,(447,63),63,(0,0,255),-1)
font = cv2.FONT_HERSHEY_SIMPLEX
cv2.putText(img,'FuXaingyang',(10,500),font,2,(255,255,255),2,cv2.LINE_AA)

plt.imshow(img[:,:,::-1])
plt.title('匹配结果'),plt.xticks([]),plt.yticks([])
plt.show()

1.3 获取并修改图像中的像素点

通过行和列的坐标值获取该像素点的像素值。

对于BRG图像，返回一个蓝绿红值的数组。

对于灰度图像，仅返回相应的强度值。

import numpy as np
import cv2
img = cv2.imread('1.jpg')
# 获取像素点的值
px = img[100,100]
# 获取蓝色通道的值
blue = img[100,100,0]
# 修改某个位置的像素值
img[100,100] = [255,255,255]

1.6 色彩空间的改变

常用BGR<->Gray和BGR<->HSV

API:

cv2.cvtColor(img,flag)

参数：

  • img：进行颜色空间转换的图像
  • flag：转换类型

    • cv2.COLOR_BGR2HSV ：BGR<->HSV

2. 算术操作

2.1 图像的加法

可以使用OpenCV的cv2.add()函数把两幅图像相加，或者通过numpy操作添加两个图像，两个图像应该具有相同的大小和类型，或者第二个图像是标量值。

OpenCV加法是饱和操作，numpy加法是模运算

2.2 图像的混合

也是加法，不同的是两幅图像的权重不同，这就会给人一种混合或者透明的感觉。

参考代码：

import numpy as np
import cv2
import matplotlib.pyplot as plt

img1 = cv2.imread('1.jpg')
img2 = cv2.imread('2.jpg')
# 图像混合
img3 = cv.addWeighted(img1,0.7,img2,0.3,0)

三. OpenCV图像处理

1. 几何变换

1.1 图像缩放

API:

cv2.resize(img,dsize,fx=0,fy=0,interpoloation=cv2.INTER_LINEAR)

参数：

  • img：输入图像

  • dsize：绝对尺寸，直接指定调整后图像的大小

  • fx，fy：相对尺寸，将dsize设置为None，然后将fx和fy设置为比例因子即可

  • interpolation：插值方法

    | 插值 | 含义 |
    | :———————-: | :——————————: |
    | cv2.INTER_LINEAR | 双线性插值法 |
    | cv2.INTER_NEAREST | 最近邻插值 |
    | cv2.INTER_AREA | 像素区域重采样（默认） |
    | cv2.INTER_CUBIC | 双三次插值 |

示例：

import cv2
img = cv2.imread('1.jpg')
rows,cols = img.shape[:2]
res1 = cv2.resize(img,(2*cols,2*rows),interpolation=cv.INTER_CUBIC)
res2 = cv2.resize(img,None,fx=0.5,fy=0.5)

1.2 图像平移

API：

cv2.warpAffine(img,M,dszie)

参数：

  • img：输入图像

  • M：2 * 3 移动矩阵

    对于处的像素点，要把它移动到处时，M矩阵应如下设置：

​ 将M设置为np.float32类型的numpy数组

  • dsize：输出图像的大小

    输出图像的大小应该是（宽度，高度）的形式

示例：

import numpy as np
import cv2
import matplotlib.pyplot as plt

img1 = cv2.imread('1.jpg')
rows,cols = img1.shape[:2]
M = np.float32([[1,0,100],[0,1,50]])
dst = cv2.warpAffine(img1,M,(cols,rows))

1.3 图像旋转

API：

cv2.getRotationMatrix2D(center, angle, scale)

参数：

  • center：旋转中心

  • angle：旋转角度

  • scale：缩放比例

返回值：

  • M：旋转矩阵

    调用cv2.warpAffine完成图像的旋转

示例：

import numpy as np
import cv2
import matplotlib.pyplot as plt
# 1 读取图像
img = cv2.imread("1.jpg")

# 2 图像旋转
rows,cols = img.shape[:2]
# 2.1 生成旋转矩阵
M = cv2.getRotationMatrix2D((cols/2,rows/2),90,1)
# 2.2 进行旋转变换
dst = cv2.warpAffine(img,M,(cols,rows))

# 3 图像展示
fig,axes=plt.subplots(nrows=1,ncols=2,figsize=(10,8),dpi=100)
axes[0].imshow(img1[:,:,::-1])
axes[0].set_title("原图")
axes[1].imshow(dst[:,:,::-1])
axes[1].set_title("旋转后结果")
plt.show()

1.4 仿射变换

示例：

import numpy as np
import cv2
import matplotlib.pyplot as plt
# 1 图像读取
img = cv2.imread("2.jpg")

# 2 仿射变换
rows,cols = img.shape[:2]
# 2.1 创建变换矩阵
pts1 = np.float32([[50,50],[200,50],[50,200]])
pts2 = np.float32([[100,100],[200,50],[100,250]])
M = cv2.getAffineTransform(pts1,pts2)
# 2.2 完成仿射变换
dst = cv2.warpAffine(img,M,(cols,rows))

# 3 图像显示
fig,axes=plt.subplots(nrows=1,ncols=2,figsize=(10,8),dpi=100)
axes[0].imshow(img[:,:,::-1])
axes[0].set_title("原图")
axes[1].imshow(dst[:,:,::-1])
axes[1].set_title("仿射后结果")
plt.show()

1.5 透射变换

示例：

import numpy as np
import cv2
import matplotlib.pyplot as plt
# 1 读取图像
img = cv2.imread("2.jpg")
# 2 透射变换
rows,cols = img.shape[:2]
# 2.1 创建变换矩阵
pts1 = np.float32([[56,65],[368,52],[28,387],[389,390]])
pts2 = np.float32([[100,145],[300,100],[80,290],[310,300]])

T = cv2.getPerspectiveTransform(pts1,pts2)
# 2.2 进行变换
dst = cv2.warpPerspective(img,T,(cols,rows))

# 3 图像显示
fig,axes=plt.subplots(nrows=1,ncols=2,figsize=(10,8),dpi=100)
axes[0].imshow(img[:,:,::-1])
axes[0].set_title("原图")
axes[1].imshow(dst[:,:,::-1])
axes[1].set_title("透射后结果")
plt.show()

1.6 图像金字塔

API：

cv2.pyrUp(img)   # 对图像进行上采样
cv2.pyrDown(img) # 对图像进行下采样

示例：

import numpy as np
import cv2
import matplotlib.pyplot as plt
# 1 图像读取
img = cv2.imread("2.jpg")
# 2 进行图像采样
up_img = cv2.pyrUp(img)  # 上采样操作
img_1 = cv2.pyrDown(img)  # 下采样操作
# 3 图像显示
cv2.imshow('enlarge', up_img)
cv2.imshow('original', img)
cv2.imshow('shrink', img_1)
cv2.waitKey(0)
cv2.destroyAllWindows()
